Our service

To simplify indoor wayfindings at large campuses like hospitals, universities, shopping malls etc., We build an Android app for digitalising a venue.

The app allows users to view interactive floor map of entire campus. They can easily search for any room, office, lab or any other entity.

Our uniqueness

Following are the multitude of features we provide :

  • View entire floor map of any building

To view the entire floor map, user just has to select the building name and floor number

  • Filter floor map based on categories

The rooms are categorised for visitor’s convenience. They can filter the floor map and see labels only for those rooms which are relevant to them

  • Easy searching across entire campus

User can search for any room within the campus

  • Alternate names for rooms

Some rooms might have multiple names. No problem, we have got you covered. You can save alternate names for rooms using which users can even search

  • Additional info about a room can be added

Additional info like contact details, working hours etc can also be added for rooms

  • Images for general entities

For convenience, we show images instead of labels for general entities like Restroom, Stairs, Elevators, Entrance/Exit, Emergency Exits etc.

  • Share indoor location via unique links

Each room has a unique link which can be shared.

  • Role based map access

    Share map of confidential area with only a specific group of people

  • Easy and fast updates to the map

You can quickly update the floor map from our easy to use Admin app
